## Features

- **3 Sizes**: sm, md, lg
- **22 Colors**: Full Tailwind palette with dark mode support
- **Adornments**: Start and end adornments for icons or quick actions
- **Helper + Error**: Mutually exclusive helper and error text
- **Full Width**: Expand to container width
- **Password**: Optional visibility toggle
- **Transparent**: Remove background for overlay layouts

## TypeScript

### InputProps Interface

```tsx
interface InputProps extends React.ComponentProps<"input"> {
  helperText?: string;
  errorText?: string;
  label?: string;
  fullWidth?: boolean;
  disabled?: boolean;
  startAdornment?: React.ReactNode;
  endAdornment?: React.ReactNode;
  color?: ColorType;
  removeBackground?: boolean;
  inputSize?: "sm" | "md" | "lg";
}

type PasswordFieldProps = Omit<InputProps, "type" | "endAdornment"> & {
  showVisibilityToggle?: boolean;
};

type ColorType =
  | "slate"
  | "gray"
  | "zinc"
  | "neutral"
  | "stone"
  | "red"
  | "orange"
  | "amber"
  | "yellow"
  | "lime"
  | "green"
  | "emerald"
  | "teal"
  | "cyan"
  | "sky"
  | "blue"
  | "indigo"
  | "violet"
  | "purple"
  | "fuchsia"
  | "pink"
  | "rose";
```

## Basic Usage

<Canvas of={InputStories.BasicUsageExample} sourceState="none" />

```tsx
import { TextField } from "@adamhinckley/design-system";

export function Example() {
  return (
    <TextField
      type="email"
      label="Email"
      helperText="We will never share your email."
    />
  );
}
```
