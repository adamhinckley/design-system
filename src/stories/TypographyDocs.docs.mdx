## Features

- **14 Variants**: h1-h6, subtitle1-2, body1-2, button, caption, overline, inherit
- **MUI-Compatible API**: Familiar variant prop for easy migration
- **22 Colors**: Full Tailwind palette with dark mode support
- **Custom Elements**: Override default HTML element per variant
- **Comprehensive Styling**: Font family, size, weight, spacing, alignment, decoration, transform
- **Truncation**: Line clamp and text overflow utilities
- **Typography Utilities**: Letter spacing, line height, text transform, word break, hyphens

## Installation

```bash
npm install @adamhinckley/design-system
```

### Available Colors

All colors automatically use:

- **Light mode**: `text-{color}-700`
- **Dark mode**: `text-{color}-300`

Colors: slate (default), gray, zinc, neutral, stone, red, orange, amber, yellow, lime, green, emerald, teal, cyan, sky, blue, indigo, violet, purple, fuchsia, pink, rose

## Custom Component Override

## Advanced Properties

### Font Properties

- **fontFamily**: "sans" | "serif" | "mono"
- **size**: "xs" through "9xl" (overrides variant default)
- **weight**: "thin" through "black"
- **italic**: boolean
- **smoothing**: "antialiased" | "subpixel-antialiased"
- **numericVariant**: Advanced numeric font variants

### Spacing & Layout

- **tracking**: Letter spacing from "tighter" to "widest"
- **leading**: Line height from "none" to "loose" or numeric values
- **lineClamp**: Limit text to specific number of lines
- **indent**: Text indentation

### Alignment

- **align**: Horizontal text alignment
- **verticalAlign**: Vertical alignment for inline elements

### Typography Utilities

- **decoration**: underline, overline, line-through
- **decorationColor**: Color of text decoration
- **decorationStyle**: solid, double, dotted, dashed, wavy
- **transform**: uppercase, lowercase, capitalize
- **overflow**: truncate, ellipsis, clip
- **wrap**: Control text wrapping behavior
- **whitespace**: Control whitespace collapsing
- **wordBreak**: Control word breaking behavior
- **hyphens**: Control hyphenation

## Default Element Mapping

Each variant has a sensible default HTML element:

- **h1-h6** → `<h1>` through `<h6>`
- **subtitle1, subtitle2** → `<h6>`
- **body1, body2** → `<p>`
- **button, caption, overline, inherit** → `<span>`

Override with the `component` prop when needed.
