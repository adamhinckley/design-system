# Design System

This repository contains a React + TypeScript design system. It ships reusable UI components, shared styles, and Storybook documentation for visual review and accessibility checks.

## Getting started

```sh
pnpm install
```

## Storybook

View the deployed Storybook: https://design-system-hazel-mu.vercel.app/

Run Storybook locally:

```sh
pnpm storybook
```

Build the static Storybook site:

```sh
pnpm build-storybook
```

## Other commands

```sh
pnpm dev
pnpm build
pnpm lint
pnpm preview
```
